# Live Object Recognition AI

This project uses YOLOv8, OpenCV, and pyttsx3 to detect objects from a live webcam feed and speak the detected object names.

## How to Run
1. Install Python 3.10+
2. Install dependencies:
# pip install ultralytics opencv-python pyttsx3
3. Run:
# python live_object_voice.py

Press `Q` to quit the program.

## Features
- Live object detection
- Real-time text-to-speech
